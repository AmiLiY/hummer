# 利益相关者
1. **平台管理员**：管理员主要负责对平台的管理和维护，以及系统的核心资源进行管理；
2. **平台用户**：软件开发者，使用平台进行项目托管和应用部署；
3. **第三方用户**：应用的使用者，通过网络访问平台上的应用。

# 关键业务概念
1. **项目**：跟实际软件开发中的“项目”概念对应；
2. **镜像**：应用在平台上托管的唯一格式；
3. **应用**：与实际开发的“应用”概念对应，也指启动之后的镜像；
4. **容器**：应用在平台上的运行形式；
5. **资源模板**：用于控制应用对资源（cpu和mem）的使用；
6. **内/外部IP地址**：应用通过内/外部地址提供服务；
7. **端口**：应用通过开放端口提供服务；
8. **环境变量**：用于向容器内部传递参数；
9. **存储卷**：持久化存储卷，类似于磁盘。
